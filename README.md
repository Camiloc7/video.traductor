# Video Subtitle Translator

Script en Python que extrae subtítulos desde video usando OCR y los traduce automáticamente.

## Características

- Detección de subtítulos quemados en video
- Extracción usando OCR
- Sistema de cache para evitar reprocesar
- Traducción automática
- Manejo de progreso

## Uso

```bash
python extractor_subtitulos.py
python traductor_video.py
